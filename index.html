HTML Documentation

HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages and web applications. It is used to structure and format the content on the web, including text, images, and videos, and it allows web browsers to interpret and display the content.

HTML is written using tags, which are enclosed in angle brackets. These tags tell the browser how to format the content and where to display it on the page. There are a wide variety of HTML tags, each with its own specific purpose and meaning. For example, the <p> tag is used to define a paragraph, the <img> tag is used to include an image, and the <a> tag is used to create a link.

Here is a simple example of an HTML document:

<html>
<head>
<title>My Web Page</title>
</head>
<body>
<h1>Welcome to my web page!</h1>
<p>This is a simple web page.</p>
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
<p><a href="https://www.example.com">Visit my website</a> for more information.</p>
</body>
</html>

This HTML document has a <head> element, which contains metadata about the document, and a <head> element, which contains the actual content of the page. The <h1> tag is used to define a heading, the <p> tag is used to define a paragraph, and the <ul> and <li> tags are used to create an unordered list. The <a> tag is used to create a link to another web page.

HTML is a powerful and flexible language that is essential for creating and developing websites and web applications. It is the foundation of the modern web and is used in conjunction with other technologies, such as CSS and JavaScript, to create rich and interactive user experiences.

HTML Tags

HTML tags are used to mark up the content on a web page. They indicate the structure and meaning of the content, and help browsers to render the content correctly.

HTML tags are enclosed in angle brackets, and typically come in pairs: an opening tag and a closing tag. The closing tag has a forward slash before the tag name. For example:

<p>This is a paragraph.</p>

In this example, <p> is the opening tag and </p> is the closing tag. The content of the paragraph, "This is a paragraph," appears between the two tags.

There are many different HTML tags that can be used to mark up a web page. Some of the most common include:

  • <html> : This tag indicates the start of an HTML document.
  • <head> : This tag contains information about the document, such as the title and any external style sheets or scripts.
  • <title> : This tag specifies the title of the document, which is displayed in the browser's title bar or tab.
  • <body> : This tag indicates the start of the document's main content.
  • <h1> to <h6> : These tag defines the headings, with <h1> being the most important and <h6> beaing the least imaportant.
  • <p> : This tag defines a paragraph.
  • <br> : This tag inserts a line break.
  • <em> : This tag italicizes text.
  • <strong> : This tag boldface text.
  • <img> : This tag inserts an image.
  • <a> : This tag creates a hyperlink.

There are many other HTML tags that you can use to create more advance pages. You can find a complete list of HTML tags at the following URL: http://www.w3schools.com/tags/.

Empty Elements

Empty elements are HTML elements that have no content and close themselves in the opening tag. They are also known as void elements or singleton elements.

Empty elements do not have an end tag in HTML instead, they are written with a start tag, and then a slash before the closing angle bracket.

Some common empty elements include:

  • <br> : inserts a line break
  • <hr> : inserts a horizontal rule (a line that runs across the page)
  • <img> : inserts an image
  • <input> : creates an input field
  • <meta> : provides the metadata(additional information about the page that can be used by the browsers) about the document
  • <link> : links to external resources, such as stylesheets

Here's an example of an empty element:

<img src="image.jpg" alt="A description of the image">

Empty elements do not have a closing tag, so the element above would be considered "empty."

Doctype Declaration

In HTML, the doctype declaration is used to specify the type of the document. It is placed at the very beginning of an HTML document, before the HTML element, and consists of a single line of code. The doctype declaration is not an HTML element and does not require a closing tag.

The doctype declaration is important because it tells the web browser how to interpret the HTML code in the document. It specifies which version of HTML the document is written in, and this information is used to ensure that the browser renders the content correctly.

Here is an example of the doctype declaration for HTML5:

<!DOCTYPE html>

This doctype declaration is used for all HTML documents written in the HTML5 specification.

In older version of HTML, the doctype declaration was longer and more complex. For example, here is the doctype declaration for HTML4.01:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

It is generally recommended to use the HTML5 doctype declaration, as it is the most current and widely supported version of HTML.

HTML Attributes

HTML attributes provide additional information about HTML elements. They are specified in the opening tag of an element and are used to configure or modify the element in some way.

Attributes are made up of a name and a value, separated by an equals sign. The value should always be enclosed in quotation marks.

For example, the href attribute is used to specify a link for an a element:

<a href="https://www.example.com">Click here</a>

Here, the attribute name is href and the attribute value is "https://www.example.com".

Other common attributes include class, which is used to apply CSS styles to an element, and id, which is used to uniquely identify an element on a page.

<div class="red-text" id="main-content">This text will be red and have an id of "main-content"</div>

Attributes can also be used to specify the dimensions of elements, such as the width and height of an image:

<img src="image.jpg" width="200" height="100">

There are many other attributes available in HTML, and the specific attributes that are available for a given element depend on the element type. You can find more information about the different attributes that are available in the HTML specification.
